## Responsive Design

The application adapts to different screen sizes:
- Desktop: Full sidebar + main panel
- Tablet: Collapsible sidebar
- Mobile: Stacked layout (not optimal for complex visualizations)

## Theme

The application uses `theme_minimal()` from ggplot2:
- Clean, uncluttered appearance
- Focus on data, not decorations
- Professional look
- Good for presentations and reports
